# 3D Library

An interactive 3D model library built with Next.js and Three.js. Browse, upload, and explore 3D models in a real-time WebGL viewer directly in your browser. The app features an admin panel for managing the model collection and a clean, responsive UI powered by shadcn/ui and Tailwind CSS. Model metadata is stored in a lightweight JSON data layer — no database required.

## Install

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Dependencies

| Package | Purpose |
|---|---|
| `next` | App framework |
| `react` / `react-dom` | UI rendering |
| `three` | 3D engine |
| `@react-three/fiber` | React renderer for Three.js |
| `@react-three/drei` | Three.js helpers & controls |
| `tailwindcss` | Utility-first styling |
| `typescript` | Type safety |
